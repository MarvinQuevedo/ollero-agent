# Ollero Project Summary

Ollero is a local code agent built using Rust and powered by Ollama.  The project leverages a variety of crates to handle HTTP requests, asynchronous operations, JSON processing, error management, terminal interaction, and more.

## Key Technologies

*   **Rust:** The primary programming language.
*   **Ollama:**  The underlying language model provider.
*   **reqwest:** For making HTTP requests.
*   **tokio:** For asynchronous runtime.
*   **serde/serde\_json:** For JSON serialization and deserialization.
*   **crossterm:** For terminal manipulation.
